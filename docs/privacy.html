TabWipe Privacy Policy

Last updated: March 1, 2026

Summary

TabWipe does not collect, store, or transmit any user data. Period.

Data Collection

TabWipe collects zero data. We have no servers, no analytics, no tracking, no telemetry. The extension operates entirely within your browser.

Data Storage

All settings (whitelist, rules, statistics) are stored locally in your browser using Chrome's built-in storage APIs (chrome.storage.local and chrome.storage.sync). Synced data goes through Chrome's own sync infrastructure — we never see it.

Permissions

  • cookies: Required to read and delete cookies (the core purpose of the extension)
  • tabs: Required to detect when tabs are opened and closed
  • storage: Required to save your whitelist and settings
  • alarms: Required for reliable background scheduling
  • host_permissions (<all_urls>): Required to access cookies for all websites

Third Parties

TabWipe does not share data with any third parties because there is no data to share.

The only external network request TabWipe may make is license key validation (Pro users only), which sends only the license key to our validation endpoint.

Open Source

TabWipe is open source. You can verify all of the above by reading the code: github.com/4ier/tabwipe
